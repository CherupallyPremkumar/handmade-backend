### Viewing Diagrams

You can view PlantUML diagrams using:

1. **VS Code Extension**: Install "PlantUML" extension
2. **Online Viewer**: Copy content to http://www.plantuml.com/plantuml/uml/
3. **Command Line**: `plantuml diagram.puml` (requires PlantUML installation)
